# 💀 المُتحكِّم V6 - وضع الوحش (Beast Mode) 💀

تم تحويل هذا البوت إلى ترسانة تكنولوجية متكاملة مصممة للسيطرة والردود الذكية الشرسة، مع توافق كامل مع تطبيق **Termux**.

---

## 🚀 المميزات الجديدة في V6
- **الهجوم المتسلسل (Sequential Attack)**: ميزة "قصف" الآن ترسل 5 عبارات مدمرة متتالية.
- **الذكاء الاصطناعي (AI)**: محرك ردود ذكي وشرس لا يرحم الأعداء.
- **نظام JSON الذكي**: استبدال SQLite بنظام JSON لضمان العمل في Termux دون مشاكل برمجية.
- **المحاكاة البشرية**: البوت يظهر كأنه يكتب (Typing...) لتجنب الحظر.
- **ترسانة شتائم ضخمة**: تمت إضافة مئات العبارات الجديدة والمحدثة.

---

## 🛠️ متطلبات التشغيل في Termux
افتح تطبيق Termux ونفذ الأوامر التالية بالترتيب:

### 1. تحديث النظام وتثبيت المتطلبات
```bash
pkg update && pkg upgrade -y
pkg install nodejs git -y
```

### 2. تحميل البوت
```bash
git clone https://github.com/seiferfanerfan-jpg/whatsapp-bot-v6.git
cd whatsapp-bot-v6
```

### 3. تثبيت المكتبات (هام جداً)
```bash
npm install @whiskeysockets/baileys pino qrcode-terminal openai dotenv @hapi/boom
```

### 4. تشغيل البوت
```bash
node index.js
```

## 📸 ربط الحساب (QR Code)
عند تشغيل البوت، سيظهر رمز QR:
1. افتح واتساب على هاتفك.
2. اذهب إلى **الأجهزة المرتبطة**.
3. اختر **ربط جهاز** وامسح الرمز الظاهر في Termux.
